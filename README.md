# Build your prompt

Локальная библиотека интерактивных генераторов промптов в едином акварельном дизайне.

## Запуск

Откройте `index.html` в браузере. Сборка и сервер не требуются.

## Генераторы

### Образовательные

- `poster.html` — учебная инфографика;
- `web.html` — интерактивный web-эквивалент материала;
- `latex_code_based.html` — LaTeX-пособие;
- `excersizes_based_on_user_choice.html` — набор упражнений;
- `complex_web_handook_based_on_users_choice.html` — комплексное многостраничное web-пособие;
- `hometask.html` — домашняя, контрольная или диагностическая работа.

### AgentKit

- `agentkit-review.html` — полное read-only ревью репозитория;
- `agentkit-fix.html` — устранение подтверждённых findings;
- `agentkit-architecture.html` — проектирование архитектуры сервиса.

AgentKit-генераторы поддерживают развёрнутый и краткий боевой режимы. Работа с Graphify во всех шаблонах ограничена корневым `./graph.json`.

## Общий функционал

- единая навигация;
- адаптивный акварельный интерфейс;
- локальное сохранение черновика формы;
- валидация обязательных полей;
- копирование с fallback для локального `file://`-режима;
- скачивание промпта в `.txt`;
- отсутствие CDN и внешних runtime-зависимостей;
- печатные стили и поддержка `prefers-reduced-motion`.

## Проверка

Требуется Node.js 22+:

```bash
node scripts/validate-pages.mjs
```

Проверка контролирует наличие страниц и shared assets, локальные ссылки, обязательные элементы форм, дублирующиеся `id`, внешние зависимости и синтаксис встроенного JavaScript. Тот же smoke-test запускается workflow `.github/workflows/validate-static.yml`.
